# Моя коллекция сниппетов для WordPress

Сборник полезных кусков PHP-кода, которые я часто использую в проектах на WordPress. 
Все это добро кидается в файл `functions.php` **дочерней темы** (child theme).

**Внимание!** Один кривой символ в `functions.php` может положить весь сайт. Всегда делайте бэкап перед редактированием!

## Что тут есть?

-   **Отключение админ-панели:** Скрывает черную полосу сверху для обычных пользователей.
-   **Свой логотип на странице входа:** Мелочь, а клиентам нравится.
-   **Регистрация кастомного типа записей:** Маст-хэв для создания портфолио, каталогов и т.д.
